# Install
- Download the zip > unzip
- with terminal cd into the folder
- `python -m venv venv`
- `scripts venv/bin/activate`
- `pip install -r requirements.txt`

# How to use
- `scripts venv/bin/activate`
- `python main.py`
- Generated xml will be stored in xmls/prothomalo folder named with current date
